# 🎓 栈微课 - 数据结构交互式学习平台

> 国内可直接访问 | 无需代理 | 完全免费

## 📖 项目简介

这是一个完整的数据结构"栈"主题微课学习系统，包含：
- 📹 9分58秒AI数字人讲解视频（8个知识章节）
- 🎯 6个时间轴互动答题（讲完即测）
- 💻 2道C++编程练习（在线编译+自动评测）
- 🤖 AI学习助手（本地知识库+在线API）
- 📝 学习笔记功能（自动保存）
